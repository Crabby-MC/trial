# 🎮 Exora Games Website

Welcome to the official **Exora Games** website! This is a modern, responsive website for showcasing our UGC (User-Generated Content) game studio.

## ✨ Features

- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
- **Modern Aesthetics** - Purple/violet color scheme matching our brand identity
- **Interactive Elements** - Smooth animations, hover effects, and scroll transitions
- **Fast Performance** - Pure HTML, CSS, and JavaScript (no dependencies)
- **Accessible** - Semantic HTML and WCAG compliant
- **SEO Optimized** - Proper meta tags and semantic structure

## 📁 File Structure

```
├── index.html        # Main website structure
├── styles.css        # All styling and animations
├── script.js         # Interactive functionality
└── README.md         # This file
```

## 🎨 Website Sections

1. **Navigation** - Sticky header with smooth scrolling
2. **Hero Section** - Eye-catching introduction with animated logo
3. **About Us** - Mission, vision, and values
4. **Games** - Showcase of game titles
5. **Features** - Why choose Exora Games
6. **Stats** - Impressive numbers with counter animation
7. **Contact** - Social links and contact form
8. **Footer** - Quick links and legal info

## 🚀 Getting Started

1. Clone this repository
2. Open `index.html` in your web browser
3. Customize the content with your actual game titles, links, and information

## 🎨 Customization

### Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #9945FF;      /* Main purple */
    --primary-dark: #7B2FD9;       /* Darker shade */
    --primary-light: #B366FF;      /* Lighter shade */
    --dark-bg: #0F0E17;            /* Dark background */
    --light-bg: #1A191F;           /* Light background */
    --text-primary: #FFFFFF;       /* Main text */
    --text-secondary: #B8B5C8;     /* Secondary text */
}
```

### Content
Simply edit the text in `index.html`:
- Update game titles and descriptions in the Games section
- Replace placeholder links with your Discord, Twitter, YouTube, etc.
- Add your email for the contact form
- Update stats with real numbers (10K+ creators, 50M+ players, etc.)

## 📱 Responsive Breakpoints

- **Desktop**: 1200px and up
- **Tablet**: 768px - 1199px
- **Mobile**: Below 768px

## ✨ Interactive Features

- **Smooth Scrolling** - Click navigation links for smooth scroll to sections
- **Mobile Menu** - Hamburger menu for mobile devices
- **Hover Effects** - Cards and buttons have interactive hover animations
- **Fade-in Animations** - Elements fade in as they come into view
- **Counter Animation** - Stats count up when section is visible
- **Form Submission** - Contact form with success feedback

## 🔧 Development Tips

### Adding a New Section
1. Add the HTML structure in `index.html`
2. Style it in `styles.css` with matching variables
3. Add any interactivity in `script.js`

### Changing the Logo
Replace the SVG code in the navigation and hero section with your actual logo. The current logo uses the Exora Games purple color (#9945FF).

### Deploying
- **GitHub Pages**: Push to your repository and enable Pages in settings
- **Vercel/Netlify**: Connect your GitHub repo and deploy automatically
- **Traditional Hosting**: Upload the files via FTP/SFTP
